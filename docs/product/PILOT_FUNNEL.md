# Pilot Funnel

Candidate
→ Contacted
→ Responded
→ Interested
→ Installed
→ Required Check Enabled
→ Retained Usage
→ Dependency Confirmed

## Metrics

- Candidates identified
- Contacted
- Response rate
- Install rate
- Required-check adoption rate
- Retention rate
- Dependency confirmations

## Dependency Criterion

The pilot succeeds only when an independent operator states:

"Removing Merge Guard would make our workflow materially worse."
