## Simple Explanation

Most systems treat reviews as comments.

MindShift treats review as structured governance evidence.

PREO converts:

```text
human review
→ machine-readable legitimacy evidence
```

## What PREO Captures

- reviewer authority
- review decision
- governance evidence
- mutation scope
- exact reviewed object
- replay-safe lineage

## Core Rule

```text
Code review alone
is not execution legitimacy.
```
