# Distributed Legitimacy Sandbox

## Purpose

This sandbox exists to model legitimacy failure behavior under distributed disagreement.

It is intentionally:
- non-production
- failure-oriented
- replay-focused
- reconciliation-focused
- topology-observability-oriented

## Initial Simulation Domains

- partition simulation
- replay race simulation
- stale replica simulation
- revocation lag simulation
- lineage divergence simulation

## Core Invariant

No valid continuity lineage
→ no valid authority
→ no valid execution

## Goal

Observe:
- deterministic convergence
- legitimacy drift
- reconciliation failure
- replay ambiguity
- lineage corruption
- revocation propagation behavior

Else:
NULL
