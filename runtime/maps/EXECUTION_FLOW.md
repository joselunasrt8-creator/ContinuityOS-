# Execution Flow

Status: Non-Operative

Core invariant:

```text
If no valid object exists → nothing happens
```

Canonical flow:

```text
Cognition
→ Input Shaping
→ ATAO
→ Authority Binding
→ AEO Compilation
→ Ω Validator
→ Policy Evaluation
→ Execution Boundary
→ Execution
→ Proof-of-Transfer
→ Registry Persistence
→ Observability
→ Reality
```

Boundary rule:

```text
Execution allowed only if VALID, AUTHORIZED, UNUSED, POLICY_VALID.
Else → NULL.
```

Exact-object rule:

```text
validated_object == executed_object
```

No alternate execution path is valid.
